## Start Bot 
- Fill in the blanks correctly
```dotenv
DISCORD_TOKEN=
DISCORD_PREFIX=
DISCORD_AUTHOR_ID=
DISCORD_CLIENT_ID=
DISCORD_PUBLIC_KEY=
DISCORD_CLIENT_SECRET_KEY=
```
- Update `.env-example` to `.env`
- Install required Packpages
```bat
npm i
```
- Start
```bat
npm run start
```
## Debug with VSCode (Windows)
* `F5` -> start
* `Shift + F5` -> stop
* `Ctrl + Shift + B` -> run build task
* `F9` -> toggle breakpoint
* `F10` -> step over
* `F11` -> step into
* `Shift + F11` -> step out
